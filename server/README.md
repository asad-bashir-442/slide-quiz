The backend for SlideQuiz. Built with Fastify and Socket.IO.

## Development

Install the dependencies and copy `.env.example` to `.env`.

```sh
pnpm run dev # Run (development)
pnpm run format # Format the codebase
```

## Production

```sh
pnpm start # Run (production)
```
