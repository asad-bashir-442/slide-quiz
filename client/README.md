The frontend for SlideQuiz. Built with React and DaisyUI.

## Development

Install the dependencies and copy `.env.example` to `.env`.

```sh
pnpm run dev # Run (development)
pnpm run lint # Lint the codebase for errors
pnpm run format # Format the codebase
```

## Production

```sh
pnpm run build # Build for production
pnpm run preview # Run built code for testing purposes
```
